Array reduce() Method

The reduce method in JavaScript is used to reduce the array to single value and executes aprovided function for each value of the array(from left to right) and the return value of the function is stored in accumulator.

Syntax

array.reduce( function (total, currentValue, Index, array), initialValue );

Example

[10, 20, 30, 40, 50, 60]

Example 2

[2.6, 20.3, 11.1, 40.7]
